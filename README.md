# 🌐 Aman Mohanty – Portfolio Website

🚀 A modern, responsive portfolio website showcasing my projects, skills, and experience as a Full Stack Web Developer.

🔗 **Live Demo:** https://your-live-link.com

---

## 📌 About The Project

This portfolio website is designed to present my technical skills, projects, and achievements in a clean and interactive way. It focuses on modern UI/UX, responsiveness, and smooth user experience.

The project demonstrates my ability to build real-world web applications using core web technologies without relying on frameworks.

---

## ✨ Features

* 🎨 Modern and clean UI/UX design
* 🌙 Dark / Light mode toggle (with persistence)
* 📱 Fully responsive (mobile, tablet, desktop)
* ⚡ Smooth scroll and reveal animations
* 📊 Scroll progress indicator
* 📂 Project filtering system
* 🧠 Interactive project modal view
* 📬 Contact form interface
* 🔗 Social media integration

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Custom CSS (Flexbox, Grid)
* **Concepts Used:**

  * DOM Manipulation
  * Event Handling
  * Local Storage
  * Intersection Observer API


Then open `index.html` in your browser.
